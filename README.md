
# Cop Morty

Sample discord bot in Python with a few events.


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  py -3 -m pip install -U discord.py
  py -3 -m pip install -U python-dotenv 
```

Start the server
